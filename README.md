# Capolavoro
Casello Autostradale Realizzato con Arduino

# Descrizione
Il progetto simula un casello autostradale con Arduino, utilizzando sensori a ultrasuoni (HC-SR04), servomotori, un lettore RFID, un tastierino numerico, LED, un buzzer passivo e un display LCD. Il sistema rileva l'avvicinamento di un veicolo, solleva una sbarra, richiede l'autenticazione tramite RFID e PIN, e al successo, apre una seconda sbarra, consentendo il passaggio.
